# Write_At_Your_Own_Risk
if you stop writing you lose your work. :)

## The Challenge
The rules are simple but brutal:
- You start typing.
- If you pause for **5 seconds**, the screen fades.
- Once the timer hits zero, all your text is cleared.
- The goal? Get your thoughts out without overthinking.
